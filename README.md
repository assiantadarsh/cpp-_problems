## 🛠️ How to Run the Code

To compile and run any of the C++ programs in this repository, ensure you have a C++ compiler (like GCC) installed on your system. 

1. **Clone the repository:**
   `git clone https://github.com/assiantadarsh/CPP-Problems.git`

2. **Navigate to the directory:**
   `cd CPP-Problems`

3. **Compile and Run:**
   `g++ filename.cpp -o output_name`
   `./output_name`
